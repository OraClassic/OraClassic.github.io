Oracle 10g 설치 가이드입니다.

## 📌 설치 전 준비 사항

- OS: Windows 10 또는 Linux
- 최소 메모리: 2GB 이상
- 디스크 공간: 5GB 이상
- 관리자 권한

## 🛠 설치 절차

1. Oracle 공식 사이트에서 10g 설치 파일 다운로드  
2. 압축 해제 후 `setup.exe` 실행  
3. 설치 마법사 안내에 따라 경로 및 비밀번호 설정  
4. Listener 설정 및 DB 인스턴스 이름 지정  
5. 설치 완료 후 SQL*Plus 실행 테스트
